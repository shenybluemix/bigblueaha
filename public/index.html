User On-Boarding portal of bigblue Aha! https://bigblue.aha.io

Technical Contact: Slack Channel - #bigblue-aha

  • If you just need a read-only access to Aha! e.g. to review items for OBDR meeting / to check an Epic description
    Use this link to create a none-paid seat in Aha! as self-service. This link will use your w3id/password to authenticate and will grant the user as a "Reviewer" to all products in Aha!.

  • If you are an offering manager who needs a paid seat - "contributor" to have the write access to Aha!.
    Use this portal to submit your request, which needs approval from Aha Liaison
    Paid Seat requests are tracked here

  • Join #bigblue-aha channel if you have any question about Aha! If you are not a member of the ibm-cloudplatform team, request it using these instructions
