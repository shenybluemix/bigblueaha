This Web Application provides the following endpoint for bigblue.aha.io

Technical Contact: sheny@us.ibm.com

Base URL: https://bigblueaha.w3ibm.mybluemix.net

GET /login -> Automatically create a user with "reviewer" role in Aha! through w3ID SSO

GET /ghelabels/[org]/[repo] -> Automatically create Aha! Labels in GHE repository

Example: https://bigblueaha.w3ibm.mybluemix.net/ghelabels/ibmcloud/TestAha

POST /milestones -> Create milestones(releases) for components

Example: { "components":[ { "reference_prefix": "BP" }, { "reference_prefix": "TRAN" } ], "releases":[ { "release": { "name":"TEST 2", "release_date": "2017-02-28" } }, { "release": { "name":"TEST 3", "release_date": "2017-02-30" } } ] }
